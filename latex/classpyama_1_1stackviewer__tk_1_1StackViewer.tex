\doxysubsection{Detailed Description}
Provides a GUI for displaying a TIFF stack. 


\begin{DoxyParams}{Parameters}
{\em root} & The frame in which to create the {\ttfamily \mbox{\hyperlink{classpyama_1_1stackviewer__tk_1_1StackViewer}{Stack\+Viewer}}}. None or {\ttfamily tkinter.\+Toplevel} \\
\hline
{\em image\+\_\+file} & Path of a TIFF file to open None or str\\
\hline
\end{DoxyParams}
The {\ttfamily \mbox{\hyperlink{classpyama_1_1stackviewer__tk_1_1StackViewer}{Stack\+Viewer}}} is the TIFF stack display tool in Py\+AMA.

It provides a {\ttfamily Contrast\+Adjuster}, a utility for adjusting the displayed color map. Changing the displayed color map only affects display, not the color values in the underlying {\ttfamily Stack}.

The {\ttfamily \mbox{\hyperlink{classpyama_1_1stackviewer__tk_1_1StackViewer}{Stack\+Viewer}}} is thread-\/safe and can display concurrent changes of the stack or of the ROIs via listeners.

Moreover, the {\ttfamily \mbox{\hyperlink{classpyama_1_1stackviewer__tk_1_1StackViewer}{Stack\+Viewer}}} implements a set of functions for interacting with a ROI adjuster\+:


\begin{DoxyItemize}
\item {\ttfamily \mbox{\hyperlink{classpyama_1_1stackviewer__tk_1_1StackViewer_a1ef5c961e513fe0c9d258a38bc72ed60}{Stack\+Viewer.\+start\+\_\+roi\+\_\+adjustment}}} creates, if necessary, a new ROI adjuster instance and invokes the {\ttfamily start\+\_\+adjustment} method of the ROI adjuster to start the ROI adjustment process.

The ROI adjuster should now start ROI adjustment.
\item {\ttfamily \mbox{\hyperlink{classpyama_1_1stackviewer__tk_1_1StackViewer_a2be98df8f15d1bca43e8a2f0f7a9c0f2}{Stack\+Viewer.\+stop\+\_\+roi\+\_\+adjustment}}} aborts the ROI adjustment process by calling the {\ttfamily stop\+\_\+adjustment} method of the ROI adjuster.

The ROI adjuster should now abort ROI adjustment and call {\ttfamily \mbox{\hyperlink{classpyama_1_1stackviewer__tk_1_1StackViewer_a826a3ae18c77d7f5e781dae1c5a8ef64}{Stack\+Viewer.\+notify\+\_\+roi\+\_\+adjustment\+\_\+finished}}}.

The ROI adjuster should leave the Stack, the \mbox{\hyperlink{classpyama_1_1stackviewer__tk_1_1StackViewer}{Stack\+Viewer}} and the canvas in a clean state.
\item {\ttfamily \mbox{\hyperlink{classpyama_1_1stackviewer__tk_1_1StackViewer_a826a3ae18c77d7f5e781dae1c5a8ef64}{Stack\+Viewer.\+notify\+\_\+roi\+\_\+adjustment\+\_\+finished}}} notifies the {\ttfamily \mbox{\hyperlink{classpyama_1_1stackviewer__tk_1_1StackViewer}{Stack\+Viewer}}} that ROI adjustment has finished.

The ROI adjuster should call this method when ROI adjustment is finished.
\item {\ttfamily \mbox{\hyperlink{classpyama_1_1stackviewer__tk_1_1StackViewer_a8bb05b7f8128e8b222e0f4a8b7ed18c6}{Stack\+Viewer.\+forget\+\_\+roi\+\_\+adjuster}}} aborts ROI adjustment by calling {\ttfamily \mbox{\hyperlink{classpyama_1_1stackviewer__tk_1_1StackViewer_a2be98df8f15d1bca43e8a2f0f7a9c0f2}{Stack\+Viewer.\+stop\+\_\+roi\+\_\+adjustment}}} if the adjustment is not finished yet.

Then it calls the {\ttfamily close} method of the ROI adjuster, if present, which preferably causes the ROI adjuster to close its window.

Finally, the internal reference of the {\ttfamily \mbox{\hyperlink{classpyama_1_1stackviewer__tk_1_1StackViewer}{Stack\+Viewer}}} to the ROI adjuster is cleared.
