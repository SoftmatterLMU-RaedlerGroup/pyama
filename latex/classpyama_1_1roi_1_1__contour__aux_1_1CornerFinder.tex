Return the index of the node with smallest distance to node {\ttfamily i}. 

There are three search modes\+:
\begin{DoxyItemize}
\item \char`\"{}free\char`\"{} (default mode)\+: find only nodes that have no neighbors yet
\item \char`\"{}half-\/free\char`\"{}\+: find only nodes that have zero or one neighbor
\item \char`\"{}half\char`\"{}\+: find only nodes that have one neighbor
\item \char`\"{}half-\/used\char`\"{} find only nodes that have one or two neighbors
\item \char`\"{}used\char`\"{}\+: find only nodes that have two neighbors already
\item None\+: find nearest node, regardless of number of neighbors If no neighbor is found for the search mode, {\ttfamily None} is returned.
\end{DoxyItemize}
